# DS-160 Visa Assistant (修复增强版)

一个本地运行的 DS-160 填写辅助工具，面向中国 B1/B2 申请资料整理和半自动填表。

**原项目作者：[ZACHIV](https://github.com/ZACHIV/DS160-agent)**  
*(本项目在此基础上进行了核心 Bug 修复与体验优化)*

> 非官方工具。请在提交前自行核对所有 DS-160 内容。

## 修复与优化说明 (v0.1.2-patched)

- **修复了安全背景选项无法正常保存的 Bug**：原版中安全背景中的诸多“是/否”勾选框存在字段不匹配的问题，现已完美修复，勾选后均能正确保存并自动填入。
- **修复了“下载加密文件”报错的 Bug**：重构了加密导出的底层逻辑，无需先生成明文文件即可直接加密导出表单资料，彻底解决了 `No dossier document loaded` 错误。
- **界面与体验优化**：精简了无用字段，修正了前端显示与后端校验逻辑。

## 功能特性

- 手动录入申请资料，导出统一 dossier JSON
- 导入 dossier 后按页面自动填写 CEAC DS-160
- 支持断点续填、页面识别、基础校验和本地审计日志
- 支持加密导出/导入申请资料，保障隐私安全

## 快速开始

从当前项目的 Release 页面下载 `ds160-assistant.exe`（Windows 版本），双击直接运行即可：

```bash
ds160-assistant.exe
```

程序启动后，会自动在后台启动本地服务，并自动打开以下两个窗口：
1. **本地助手网页**：`http://127.0.0.1:8765`
2. **Chrome 浏览器**：自动打开官方 CEAC 网站

*(注：需要你的电脑上已经安装过 Google Chrome 浏览器。)*

## 使用流程

1. 在**本地助手网页**（127.0.0.1）的首页，进入“填写资料”。
2. 完整填写申请人信息并点击底部“保存”，导出你的专属资料 JSON（推荐使用加密导出）。
3. 回到首页，进入“开始填写”助手页。
4. 在左侧面板点击“导入 JSON”，选择你刚才导出的文件（若加密需输入密码）。
5. 切换到弹出的 **Chrome 浏览器** 中，手动选择国家、输入验证码，并在官方网站创建或恢复一个签证申请。
6. 一直点击下一步，直到进入官方实质填表页面（例如 Personal 1）。
7. 此时本地助手会自动侦测到当前页，你只需在本地助手网页上点击 **“一键填入”**。
8. 助手会在 Chrome 中自动为你填好所有的选项。你在官方页面核对无误后点击 `Next`，然后继续在助手点击“一键填入”，直到完成所有步骤！
